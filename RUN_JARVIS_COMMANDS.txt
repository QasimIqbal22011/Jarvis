================================================================================
                         HOW TO RUN JARVIS - SIMPLE GUIDE
================================================================================

🎯 MAIN COMMAND TO RUN JARVIS:

    python tray_app.py

That's it! But you need to do a few things first...

================================================================================
                           📋 BEFORE YOU START
================================================================================

1. INSTALL OLLAMA (the AI brain)
   Download: https://ollama.ai
   
2. DOWNLOAD A MODEL
   Open command prompt and run:
   
   ollama pull llama3.1:8b
   
   OR
   
   ollama pull mistral:7b

3. INSTALL PYTHON DEPENDENCIES
   
   cd C:\Jarvis
   pip install ollama pyttsx3 pyaudio sounddevice openwakeword faster-whisper webrtcvad pystray pillow pywebview send2trash psutil

================================================================================
                      🚀 STEP-BY-STEP STARTUP (EVERY TIME)
================================================================================

STEP 1: Open Command Prompt / PowerShell

STEP 2: Navigate to Jarvis folder
   cd C:\Jarvis

STEP 3: Activate virtual environment (if you have one)
   venv\Scripts\activate

STEP 4: Start Ollama in ANOTHER terminal/PowerShell window
   ollama serve
   
   (Leave this running in the background)

STEP 5: Back in the first terminal, run Jarvis:
   python tray_app.py

STEP 6: Wait for the popup window with the blue sphere to appear

STEP 7: Say "Hey Jarvis" into your microphone

Done! 🎉

================================================================================
                        🛠️ TROUBLESHOOTING
================================================================================

PROBLEM: "No module named 'X'"
SOLUTION: Install missing package
   pip install [package_name]

PROBLEM: "Ollama is not responding"
SOLUTION: Make sure Ollama is running in another terminal
   ollama serve

PROBLEM: "Microphone not found"
SOLUTION: 
   1. Check microphone is plugged in
   2. Check Windows sound settings
   3. Allow Jarvis access in Windows Privacy settings

PROBLEM: "Permission denied"
SOLUTION: Run as administrator
   Right-click Command Prompt → "Run as administrator"

PROBLEM: Can't find the window
SOLUTION: Look in the system tray (bottom right of taskbar)
   Click the up arrow if you don't see it
