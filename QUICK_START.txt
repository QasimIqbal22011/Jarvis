# ⚡ QUICK START COMMAND REFERENCE

## The Quick Way (Copy & Paste)

### For First Time Setup:

```bash
cd C:\Jarvis
python -m venv venv
venv\Scripts\activate
pip install ollama pyttsx3 pyaudio sounddevice openwakeword faster-whisper webrtcvad pystray pillow pywebview send2trash psutil
```

### Start Ollama (In Separate Terminal):

```bash
ollama serve
```

### Run Jarvis:

```bash
cd C:\Jarvis
venv\Scripts\activate
python tray_app.py
```

## All Commands at a Glance

| Task | Command |
|------|---------|
| **Navigate to project** | `cd C:\Jarvis` |
| **Activate virtual env** | `venv\Scripts\activate` |

## If Something Goes Wrong

1. **Check Ollama is running** (do `ollama list` in another terminal)
2. **Check microphone is plugged in** and enabled in Windows
3. **Reinstall packages** if modules missing:
   ```bash
   pip install --force-reinstall -r requirements.txt
   ```
4. **Check logs** for error details:
   ```bash
   type %USERPROFILE%\jarvis_log.txt
   ```
5. **Run as admin** if permission errors:
   - Right-click Command Prompt
   - Select "Run as administrator"
